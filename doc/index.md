Getting Started
===============

Prerequisites
-------------

This library require:

- jquery
- hammerjs
- font-awesome (optional, for toggle button)
- sonatra-hammer-scroll (optional)
- bootstrap (optional)

Installation
------------

Installation is a quick:

Add library in your `bower.json`:

```json
{
    "dependencies": {
        "sonatra-jquery-sidebar: "latest"
    }
}
```

Launch bower:

```bash
$ bower install
```
